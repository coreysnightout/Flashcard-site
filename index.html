JavaScript (JS)

JavaScript is often abbreviated as "JS", is a high-level, dynamic, untyped, and interpreted run-time language.

.

Operators

The assignment operator (=) assigns a value to a variable.

Variables

JavaScript variables are containers for storing data values.

Variable naming conventions

We use camelCase for identifier names (variables and functions).

Functions

A function is a block of code designed to perform a particular task.

Methods

Methods are actions that can be performed on objects.

Arguments

Function arguments are the real values passed to (and received by) the function.

Parameters

Function arguments are the real values passed to (and received by) the function.

Return

The return statement stops the execution of a function and returns a value from that functiom.

Chaining methods

Chaining allows us to run multiple jQuery methods (on the same element) within a single statement.

Strings

A JavaScript string stores a series of characters like "John Doe".

Booleans

A JavaScript Boolean represents one of two values: true or false.

Undefined

The undefined property indicates that a variable has not been assigned a value.

NaN

The NaN property represents "Not-a-Number" value. This property indicates that a value is not a legal number.

Escape

The escape() function encodes a string.

Alert

Brings up a pop up notification on the screen.

Comments

Allows you to write messages within your code that are not run within the program.

jQuery

jQuery is a lightweight, "write less, do more", JavaScript library.

Attributes

An attribute is a variable that contains specific information used to control the target system. Attributes are often (but not necessarily) visible in the user interface.
